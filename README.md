# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1djaX5WAmxwrVGobZdukj_F_XrHhPDgby

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the Gemini API key in [.env.local](.env.local) using any of these names: `VITE_GEMINI_API_KEY`, `GEMINI_API_KEY`, `VITE_API_KEY`, or `API_KEY`
3. Run the app:
   `npm run dev`
